This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

🔹 Core Concept

* TripMate is a travel companion discovery platform
* Connects users based on compatibility (interests, budget, travel style, gender preference)
* Focuses on safe, social, and collaborative travel experiences
* Goes beyond booking apps by emphasizing human connection & trust

🔹 Working Principle

1. Users create a profile with personal and travel preferences
2. System generates compatibility-based matches
3. Users can create duo or group trips with custom conditions
4. Other users can search, filter, and send join requests
5. Trip creators accept or reject requests
6. Accepted users get access to a real-time chat for coordination
